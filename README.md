# Settl API

Backend service for Settl — group expense splitter with UPI-native settlement.

## Tech Stack
- NestJS
- PostgreSQL
- Redis
- Docker
- GitHub Actions

## Local Setup

### Prerequisites
- Docker and Docker Compose installed
- Node.js 20+
- npm

### Steps
1. Clone the repo
2. Copy `.env.example` to `.env` and fill in values
3. Run `docker compose up -d` to start Postgres and Redis
4. Run `npm install`
5. Run `npm run start:dev`
